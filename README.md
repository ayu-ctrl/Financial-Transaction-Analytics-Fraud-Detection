# Financial Transaction Analytics & Fraud Detection

## Overview

This project analyzes **5 million+ financial transactions** to identify fraud patterns, payment trends, merchant insights, and business KPIs using **Python, PostgreSQL, SQL, and Power BI**.

---

## Tech Stack

- Python
- PostgreSQL
- SQL
- Power BI
- Pandas
- Jupyter Notebook

---

## Dataset

- **5,000,000+ Transactions**
- **24 Features**
- Fraud Rate: **3.59%**

> The dataset is not uploaded because it exceeds GitHub's file size limit.

---

## Key Analyses

- Business KPI Analysis
- Merchant Category Analysis
- Payment Channel Analysis
- Location-wise Fraud Analysis
- Time-based Transaction Analysis
- Fraud Trend Analysis
- SQL Window Functions
- Ranking & Cumulative Analysis
