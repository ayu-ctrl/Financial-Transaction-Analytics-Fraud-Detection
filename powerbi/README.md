# Power BI Dashboard

This folder contains the interactive Power BI dashboard developed for financial transaction analytics and fraud detection.

## Key Performance Indicators (KPIs)

- Total Transactions
- Total Transaction Value
- Average Transaction Value
- Fraud Rate

---

## Dashboard Visualizations

### 📈 Total Transactions by Month
Tracks monthly transaction volume to identify seasonal trends.

### 🌍 Fraud Transactions by Location
Displays fraud transactions across different locations for regional analysis.

### 🛒 Fraud Transactions by Merchant Category
Highlights merchant categories with higher fraud activity.

### 💳 Total Transactions by Payment Channel
Compares transaction volume across payment methods such as Card, UPI, ACH, and Wire Transfer.

### 🔄 Transaction Type Distribution
Shows the distribution of Deposit, Payment, Transfer, and Withdrawal transactions.

---

## Interactive Filters

The dashboard includes dynamic slicers for:

- Month
- Location
- Fraud Status

These filters allow users to interactively explore the data.

---

## Tools Used

- Microsoft Power BI
- DAX
- CSV Dataset
- SQL (PostgreSQL)
- Python (Data Preparation)

---

## Business Insights

- Overall fraud rate is approximately **3.59%**.
- Transaction volume remains relatively stable throughout the year.
- Fraud cases are distributed across multiple locations.
- Payment channels exhibit similar transaction volumes.
- Merchant categories can be compared interactively using dashboard filters.
