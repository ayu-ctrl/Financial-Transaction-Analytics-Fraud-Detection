# Power BI Dashboard

This folder contains the interactive Power BI dashboard developed for financial transaction analytics and fraud detection.

Dashboard KPIs:
- Total Transactions
- Total Transaction Value
- Average Transaction Value
- Fraud Rate

Visualization:
- Monthly Transaction Trend

![](images/dashboard.png)


The dashboard was built using Microsoft Power BI.

Due to GitHub's 25 MB web upload limit, the `.pbix` file is not included in this repository.

Dashboard screenshots are available in the `images/` folder.
